Algebra (Basic)

Algebraic Identities

  • (a + b)^2 = a^2 + 2ab + b^2
  • (a - b)^2 = a^2 - 2ab + b^2
  • a^2 + b^2 = (a+b)^2 - 2ab
  • a^2 + b^2 = (a-b)^2 + 2ab
  • a^2 + b^2 = ((a+b)^2 + (a-b)^2)/2
  • ab = ((a+b)^2 - (a-b)^2)/4
  • a^2 - b^2 = (a+b)(a-b)
  • (a+b+c)^2 = a^2 + b^2 + c^2 + 2(ab + bc + ca)
  • a^2 + b^2 + c^2 = (a+b+c)^2 - 2(ab + bc + ca)
  • 2(a^2 + b^2 + c^2) = (a+b)^2 + (b+c)^2 + (c+a)^2
  • (a-b-c)^2 = a^2 + b^2 + c^2 - 2ab + 2bc - 2ac
  • (a + b)^3 = a^3 + b^3 + 3ab(a+b)
  • a^3 + b^3 = (a+b)(a^2 - ab + b^2)
  • (a - b)^3 = a^3 - b^3 - 3ab(a-b)
  • a^3 - b^3 = (a-b)(a^2 + ab + b^2)
  • x^2 + (a+b)x + ab = (x+a)(x+b)

Exponents & Powers

  • a^m × a^n = a^(m+n)
  • a^m / a^n = a^(m-n) (m ≥ n)
  • a^m / a^n = 1 / a^(n-m) (n > m)
  • (ab)^n = a^n × b^n
  • (a/b)^n = a^n / b^n, b ≠ 0
  • (a^m)^n = a^(mn)
  • a^0 = 1, a ≠ 0
  • a^(-n) = 1 / a^n, a ≠ 0
  • √a = a^(1/2)
  • ∛a = a^(1/3)

Logarithms

  • log_a 1 = 0
  • log_a a = 1
  • log_a (MN) = log_a M + log_a N
  • log_a (M/N) = log_a M - log_a N
  • log_a M^r = r log_a M
  • log_a M = log_b M × log_a b
  • log_a b = 1 / log_b a

Ratio & Proportion

  • Invertendo: If a:b = c:d, then b:a = d:c
  • Alternendo: If a:b = c:d, then a:c = b:d
  • Componendo: If a:b = c:d, then (a+b)/b = (c+d)/d
  • Dividendo: If a:b = c:d, then (a-b)/b = (c-d)/d
  • Componendo-Dividendo: If a:b = c:d, then (a+b)/(a-b) = (c+d)/(c-d)
  • Multiple Proportions: If a/b = c/d = e/f = g/h, then (a+c+e+g)/(b+d+f+h) = common ratio

Linear Systems

  • System: a1 x + b1 y + c1 = 0 , a2 x + b2 y + c2 = 0
  • Cross multiplication: x/(b1c2-b2c1) = y/(c1a2-c2a1) = 1/(a1b2-a2b1)
  • Consistency:
    • a1/a2 ≠ b1/b2 → Unique solution
    • a1/a2 = b1/b2 = c1/c2 → Infinite solutions
    • a1/a2 = b1/b2 ≠ c1/c2 → No solution
  • Homogeneous (c1=c2=0):
    • a1/a2 ≠ b1/b2 → Trivial solution (0,0)
    • a1/a2 = b1/b2 → Infinite solutions

Quadratic Equations

  • ax² + bx + c = 0
  • Quadratic formula: x = (-b ± √(b²-4ac)) / 2a
  • Discriminant: D = b² - 4ac
  • Nature of roots:
    • D > 0, perfect square → real, unequal, rational
    • D > 0, not perfect square → real, unequal, irrational
    • D = 0 → real, equal
    • D < 0 → no real roots
  • Sum of roots: α + β = -b/a
  • Product of roots: αβ = c/a
  • Formation: x² - (sum)x + product = 0

Cubic Equations (Optional)

  • ax³ + bx² + cx + d = 0, roots α,β,γ
  • Sum: α+β+γ = -b/a
  • Sum of products two at a time: αβ + βγ + γα = c/a
  • Product: αβγ = -d/a
  • Formation: x³ - (sum)x² + (sum of products)x - product = 0

Binomial Theorem & Factorials

  • (x + y)^n = Σ (nCr) x^(n-r) y^r , r = 0 to n
  • General term: T_(r+1) = nCr x^(n-r) y^r
  • Factorial: n! = n(n-1)(n-2)...2*1, 0! = 1
  • Combinations: nCr = n! / [r!(n-r)!]

Factor Theorem (Optional)

  • (x-a) is a factor of f(x) ⇔ f(a) = 0
  • (ax+b) is a factor of f(x) ⇔ f(-b/a) = 0
