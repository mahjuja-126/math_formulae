Algebra (Basic)

Algebraic Identities

  • (a + b)² = a² + 2ab + b²
  • (a − b)² = a² − 2ab + b²
  • (a + b)(a − b) = a² − b²
  • a² + b² = (a + b)² − 2ab
  • a² + b² = (a − b)² + 2ab
  • a² + b² = [(a + b)² + (a − b)²] / 2
  • ab = [(a + b)² − (a − b)²] / 4
  • (a + b + c)² = a² + b² + c² + 2ab + 2bc + 2ca
  • (a + b)³ = a³ + 3a²b + 3ab² + b³
  • a³ + b³ = (a + b)(a² − ab + b²)
  • a³ − b³ = (a − b)(a² + ab + b²)

Laws of Indices

  • aᵐ × aⁿ = aᵐ⁺ⁿ
  • aᵐ / aⁿ = aᵐ⁻ⁿ
  • (ab)ⁿ = aⁿbⁿ
  • (a/b)ⁿ = aⁿ / bⁿ (b ≠ 0)
  • (aᵐ)ⁿ = aᵐⁿ
  • a⁰ = 1 (a ≠ 0)
  • a⁻ⁿ = 1 / aⁿ

Logarithms

  • logₐ 1 = 0
  • logₐ a = 1
  • logₐ(MN) = logₐM + logₐN
  • logₐ(M/N) = logₐM − logₐN
  • logₐ(Mʳ) = r logₐM
  • logₐM = log_bM × logₐb

Quadratic Equations

  • General form: ax² + bx + c = 0
  • x = (−b ± √(b² − 4ac)) / 2a
  • Discriminant: D = b² − 4ac
  • D > 0 → real and unequal roots
  • D = 0 → real and equal roots
  • D < 0 → no real roots

Ratio and Proportion

  • If a : b = c : d, then b : a = d : c
  • If a : b = c : d, then a : c = b : d
  • (a + b)/b = (c + d)/d
  • (a − b)/b = (c − d)/d
  • (a + b)/(a − b) = (c + d)/(c − d)

Binomial Theorem

  • (x + y)ⁿ = Σ nCr · xⁿ⁻ʳyʳ
  • General term: Tᵣ₊₁ = nCr · xⁿ⁻ʳyʳ
  • n! = n(n−1)(n−2)…1
  • nCr = n! / [r!(n−r)!]
