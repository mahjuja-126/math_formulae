Integration (Intro)

Fundamental Rules

\(\int k\,dx = kx + C\)

Derivation:

Let \(F(x) = kx + C\).

Differentiating with respect to \(x\):

\[ \frac{d}{dx}(kx + C) = \frac{d}{dx}(kx) + \frac{d}{dx}(C) = k(1) + 0 = k \]

Since the derivative of \(kx + C\) is \(k\),

\[ \int k\,dx = kx + C. \]
\(\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \ne -1)\)

Derivation:

Let \(F(x) = \frac{x^{n+1}}{n+1} + C\).

Differentiating:

\[ \frac{d}{dx}\left(\frac{x^{n+1}}{n+1} + C\right) = \frac{1}{n+1}\cdot (n+1)x^{(n+1)-1} + 0 = x^n \]

Since the derivative equals \(x^n\),

\[ \int x^n dx = \frac{x^{n+1}}{n+1} + C, \quad n \ne -1. \]
\(\int \frac{1}{x} dx = \ln|x| + C\)

Derivation:

Case 1: \(x > 0\)

\[ \frac{d}{dx}(\ln x) = \frac{1}{x} \]

Case 2: \(x < 0\)

Let \(u = -x\), so \(u > 0\).

\[ \frac{d}{dx}(\ln(-x)) = \frac{1}{-x}\cdot (-1) = \frac{1}{x} \]

Thus, in both cases:

\[ \frac{d}{dx}(\ln |x|) = \frac{1}{x} \]

Therefore,

\[ \int \frac{1}{x} dx = \ln|x| + C. \]
\(\int (u \pm v)\,dx = \int u\,dx \pm \int v\,dx\)

Derivation:

Let \(U = \int u\,dx\) and \(V = \int v\,dx\).

Then:

\[ \frac{dU}{dx} = u, \qquad \frac{dV}{dx} = v \]

Using linearity of differentiation:

\[ \frac{d}{dx}(U \pm V) = \frac{dU}{dx} \pm \frac{dV}{dx} = u \pm v \]

Integrating both sides:

\[ \int (u \pm v)\,dx = U \pm V \]

Substituting back:

\[ \int (u \pm v)\,dx = \int u\,dx \pm \int v\,dx. \]

Exponential & Logarithmic Integrals

\(\int e^x dx = e^x + C\)

Derivation:

By definition of the natural exponential function,

\[ \frac{d}{dx}(e^x) = e^x \]

Therefore,

\[ \frac{d}{dx}(e^x + C) = e^x + 0 = e^x \]

Since the derivative of \(e^x + C\) is \(e^x\),

\[ \int e^x dx = e^x + C. \]
\(\int a^x dx = \frac{a^x}{\ln a} + C\)

Condition: \(a > 0,\; a \ne 1\)

Derivation:

Recall that:

\[ a^x = e^{x \ln a} \]

Let \(F(x) = \frac{a^x}{\ln a} + C\).

Differentiating:

\[ \frac{d}{dx}\left(\frac{a^x}{\ln a} + C\right) = \frac{1}{\ln a} \cdot \frac{d}{dx}(a^x) \] \[ = \frac{1}{\ln a} \cdot (a^x \ln a) = a^x \]

Hence,

\[ \int a^x dx = \frac{a^x}{\ln a} + C. \]
\(\int e^{mx} dx = \frac{1}{m}e^{mx} + C\)

Condition: \(m \ne 0\)

Derivation:

Let \(F(x) = \frac{1}{m}e^{mx} + C\).

Using the Chain Rule:

\[ \frac{d}{dx}\left(\frac{1}{m}e^{mx}\right) = \frac{1}{m} \cdot e^{mx} \cdot m = e^{mx} \]

Therefore,

\[ \int e^{mx} dx = \frac{1}{m}e^{mx} + C. \]
\(\int \ln x\,dx = x\ln x - x + C\)

Derivation (Integration by Parts):

Using the formula:

\[ \int u\,dv = uv - \int v\,du \]

Let:

\[ u = \ln x, \quad dv = dx \]

Then:

\[ du = \frac{1}{x}dx, \quad v = x \]

Applying the formula:

\[ \int \ln x\,dx = x\ln x - \int x \cdot \frac{1}{x} dx \] \[ = x\ln x - \int 1\,dx \] \[ = x\ln x - x + C \]

Basic Trigonometric Integrals

\(\int \sin(ax)\,dx = -\frac{1}{a}\cos(ax) + C\)

Derivation:

\[ \frac{d}{dx}\left(-\frac{1}{a}\cos(ax)\right) = -\frac{1}{a} \cdot (-\sin(ax) \cdot a) = \sin(ax) \]

Hence,

\[ \int \sin(ax)\,dx = -\frac{1}{a}\cos(ax) + C \]
\(\int \cos(ax)\,dx = \frac{1}{a}\sin(ax) + C\)

Derivation:

\[ \frac{d}{dx}\left(\frac{1}{a}\sin(ax)\right) = \frac{1}{a} \cdot (\cos(ax) \cdot a) = \cos(ax) \]

Thus,

\[ \int \cos(ax)\,dx = \frac{1}{a}\sin(ax) + C \]
\(\int \sec^2 x\,dx = \tan x + C\)

Derivation:

\[ \frac{d}{dx}(\tan x) = \sec^2 x \]

So,

\[ \int \sec^2 x\,dx = \tan x + C \]
\(\int \csc^2 x\,dx = -\cot x + C\)

Derivation:

\[ \frac{d}{dx}(-\cot x) = -(-\csc^2 x) = \csc^2 x \]

Therefore,

\[ \int \csc^2 x\,dx = -\cot x + C \]
\(\int \sec x \tan x\,dx = \sec x + C\)

Derivation:

\[ \frac{d}{dx}(\sec x) = \sec x \tan x \]

Thus,

\[ \int \sec x \tan x\,dx = \sec x + C \]
\(\int \csc x \cot x\,dx = -\csc x + C\)

Derivation:

\[ \frac{d}{dx}(-\csc x) = -(-\csc x \cot x) = \csc x \cot x \]

Hence,

\[ \int \csc x \cot x\,dx = -\csc x + C \]

Substitution-Type Integrals

These are obtained using substitution.

\(\int \frac{f'(x)}{f(x)} dx = \ln|f(x)| + C\)

Derivation:

Let \(u = f(x) \implies du = f'(x) dx\).

\[ \int \frac{f'(x)}{f(x)} dx = \int \frac{du}{u} = \ln|u| + C = \ln|f(x)| + C \]
\(\int f(x)^n f'(x)\,dx = \frac{f(x)^{n+1}}{n+1} + C \quad (n \ne -1)\)

Derivation:

Substitute \(u = f(x) \implies du = f'(x) dx\).

\[ \int f(x)^n f'(x) dx = \int u^n du = \frac{u^{n+1}}{n+1} + C = \frac{f(x)^{n+1}}{n+1} + C \]
\(\int \frac{f'(x)}{\sqrt{f(x)}} dx = 2\sqrt{f(x)} + C\)

Derivation:

Let \(u = f(x) \implies du = f'(x) dx\).

\[ \int \frac{f'(x)}{\sqrt{f(x)}} dx = \int u^{-1/2} du = 2 u^{1/2} + C = 2 \sqrt{f(x)} + C \]
\(\int (ax+b)^n dx = \frac{(ax+b)^{n+1}}{a(n+1)} + C \quad (n \ne -1)\)

Derivation:

Substitute \(u = ax+b \implies du = a\,dx \implies dx = \frac{du}{a}\).

\[ \int (ax+b)^n dx = \int u^n \frac{du}{a} = \frac{1}{a} \int u^n du = \frac{1}{a} \cdot \frac{u^{n+1}}{n+1} + C = \frac{(ax+b)^{n+1}}{a(n+1)} + C \]

Integration by Parts

Used when the integrand is a product of two functions.

\(\int u\,dv = uv - \int v\,du\)

Derivation:

Start with the Product Rule for derivatives:

\[ \frac{d}{dx}[u(x)v(x)] = u \frac{dv}{dx} + v \frac{du}{dx} \]

Integrate both sides with respect to \(x\):

\[ \int \frac{d}{dx}[u v]\,dx = \int u \frac{dv}{dx}\,dx + \int v \frac{du}{dx}\,dx \] \[ uv = \int u\,dv + \int v\,du \]

Rearranging for \(\int u\,dv\):

\[ \int u\,dv = uv - \int v\,du \]

This is the standard Integration by Parts formula.

Tip (LIATE rule for choosing \(u\)): Logarithmic → Inverse Trigonometric → Algebraic → Trigonometric → Exponential

Inverse Trigonometric Forms

\(\int \frac{dx}{1+x^2} = \tan^{-1}x + C\)

Derivation:

This is a direct result of the derivative:

\[ \frac{d}{dx}(\tan^{-1}x) = \frac{1}{1+x^2} \quad \implies \quad \int \frac{dx}{1+x^2} = \tan^{-1}x + C \]
\(\int \frac{dx}{a^2+x^2} = \frac{1}{a}\tan^{-1}\left(\frac{x}{a}\right) + C\)

Derivation:

Let \(u = x/a \implies x = au, dx = a\,du\). Then:

\[ \int \frac{dx}{a^2+x^2} = \int \frac{a\,du}{a^2 + a^2 u^2} = \int \frac{du}{a(1+u^2)} = \frac{1}{a} \int \frac{du}{1+u^2} = \frac{1}{a} \tan^{-1}(u) + C \] \[ \therefore \int \frac{dx}{a^2+x^2} = \frac{1}{a} \tan^{-1}\left(\frac{x}{a}\right) + C \]
\(\int \frac{dx}{a^2-x^2} = \frac{1}{2a}\ln\left|\frac{a+x}{a-x}\right| + C\)

Derivation:

Use partial fractions: \[ \frac{1}{a^2-x^2} = \frac{1}{(a-x)(a+x)} = \frac{1}{2a}\left(\frac{1}{a-x} + \frac{1}{a+x}\right) \] \[ \int \frac{dx}{a^2-x^2} = \frac{1}{2a} \int \left(\frac{1}{a-x} + \frac{1}{a+x}\right) dx = \frac{1}{2a} \ln\left|\frac{a+x}{a-x}\right| + C \]
\(\int \frac{dx}{x^2-a^2} = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C\)

Derivation:

Again using partial fractions: \[ \frac{1}{x^2-a^2} = \frac{1}{(x-a)(x+a)} = \frac{1}{2a}\left(\frac{1}{x-a} - \frac{1}{x+a}\right) \] \[ \int \frac{dx}{x^2-a^2} = \frac{1}{2a} \int \left(\frac{1}{x-a} - \frac{1}{x+a}\right) dx = \frac{1}{2a} \ln\left|\frac{x-a}{x+a}\right| + C \]
\(\int \frac{dx}{x\sqrt{x^2-1}} = \sec^{-1}x + C\)

Derivation:

Let \(y = \sec^{-1}x \implies \sec y = x \implies d(\sec y)/dx = 1/\frac{d}{dy}(\sec y) \)

\[ \frac{d}{dx}(\sec^{-1}x) = \frac{1}{x \sqrt{x^2-1}} \quad \implies \quad \int \frac{dx}{x\sqrt{x^2-1}} = \sec^{-1}x + C \]

Square Root Forms

\(\int \sqrt{a^2-x^2}\,dx = \frac{x}{2}\sqrt{a^2-x^2} + \frac{a^2}{2}\sin^{-1}\left(\frac{x}{a}\right) + C\)

Derivation:

Let \(x = a \sin\theta \implies dx = a \cos\theta\, d\theta\). Then:

\[ \int \sqrt{a^2 - x^2}\,dx = \int \sqrt{a^2 - a^2 \sin^2\theta} \, a\cos\theta\, d\theta = a^2 \int \cos^2\theta \, d\theta \] Use the power-reduction identity \(\cos^2\theta = \frac{1+\cos2\theta}{2}\): \[ a^2 \int \cos^2\theta \, d\theta = \frac{a^2}{2} \int (1 + \cos 2\theta)\, d\theta = \frac{a^2}{2} \left(\theta + \frac{1}{2}\sin 2\theta\right) + C \] Substitute back: \(\theta = \sin^{-1}(x/a)\), \(\sin 2\theta = 2\sin\theta \cos\theta = 2(x/a)\sqrt{1-(x/a)^2}\): \[ \int \sqrt{a^2-x^2}\,dx = \frac{x}{2}\sqrt{a^2-x^2} + \frac{a^2}{2} \sin^{-1}\left(\frac{x}{a}\right) + C \]
\(\int \sqrt{x^2+a^2}\,dx = \frac{x}{2}\sqrt{x^2+a^2} + \frac{a^2}{2}\ln\left|x+\sqrt{x^2+a^2}\right| + C\)

Derivation:

Let \(x = a \tan\theta \implies dx = a \sec^2\theta\, d\theta\), then \(\sqrt{x^2+a^2} = a \sec\theta\): \[ \int \sqrt{x^2+a^2}\,dx = \int a \sec\theta \cdot a \sec^2\theta \, d\theta = a^2 \int \sec^3\theta \, d\theta \] Using \(\int \sec^3\theta \, d\theta = \frac{1}{2} (\sec\theta \tan\theta + \ln|\sec\theta + \tan\theta|)\): \[ \int \sqrt{x^2+a^2}\,dx = \frac{a^2}{2} \left(\sec\theta \tan\theta + \ln|\sec\theta + \tan\theta|\right) + C \] Substitute back: \(\sec\theta = \sqrt{x^2+a^2}/a\), \(\tan\theta = x/a\): \[ \int \sqrt{x^2+a^2}\,dx = \frac{x}{2}\sqrt{x^2+a^2} + \frac{a^2}{2} \ln\left|x + \sqrt{x^2+a^2}\right| + C \]
\(\int \sqrt{x^2-a^2}\,dx = \frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\ln\left|x+\sqrt{x^2-a^2}\right| + C\)

Derivation:

Let \(x = a \sec\theta \implies dx = a \sec\theta \tan\theta\, d\theta\), then \(\sqrt{x^2-a^2} = a \tan\theta\): \[ \int \sqrt{x^2-a^2}\,dx = \int a \tan\theta \cdot a \sec\theta \tan\theta\, d\theta = a^2 \int \tan^2\theta \sec\theta\, d\theta \] Use \(\tan^2\theta = \sec^2\theta - 1\) and integrate: \[ \int \sqrt{x^2-a^2}\,dx = \frac{1}{2} x\sqrt{x^2-a^2} - \frac{a^2}{2} \ln|x + \sqrt{x^2-a^2}| + C \]

Reduction Formulas

\(\int \sin^n x\,dx = -\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x\,dx\)

Derivation:

Using Integration by Parts, let \(u = \sin^{n-1}x\), \(dv = \sin x dx\) → \(du = (n-1)\sin^{n-2}x \cos x dx\), \(v = -\cos x\): \[ \int \sin^n x dx = -\sin^{n-1}x \cos x + (n-1) \int \sin^{n-2} x dx \] Divide by \(n\) to get the standard reduction formula: \[ \int \sin^n x dx = -\frac{1}{n} \sin^{n-1}x \cos x + \frac{n-1}{n} \int \sin^{n-2} x dx \]
\(\int \cos^n x\,dx = \frac{1}{n}\cos^{n-1}x\sin x + \frac{n-1}{n}\int \cos^{n-2}x\,dx\)

Derivation:

Using Integration by Parts, let \(u = \cos^{n-1}x\), \(dv = \cos x dx\) → \(du = -(n-1) \cos^{n-2}x \sin x dx\), \(v = \sin x\): \[ \int \cos^n x dx = \cos^{n-1}x \sin x + (n-1) \int \cos^{n-2} x dx \] Divide by \(n\) to match the standard reduction formula: \[ \int \cos^n x dx = \frac{1}{n} \cos^{n-1}x \sin x + \frac{n-1}{n} \int \cos^{n-2} x dx \]
\(\int x^n e^x dx = x^n e^x - n\int x^{n-1} e^x dx\)

Derivation:

Use Integration by Parts, let \(u = x^n\), \(dv = e^x dx\) → \(du = n x^{n-1} dx\), \(v = e^x\): \[ \int x^n e^x dx = x^n e^x - \int n x^{n-1} e^x dx = x^n e^x - n \int x^{n-1} e^x dx \]
