Integration (Intro)

Fundamental Rules

\(\int k\,dx = kx + C\)
\(\int x^n dx = \frac{x^{n+1}}{n+1} + C \quad (n \ne -1)\)
\(\int \frac{1}{x} dx = \ln|x| + C\)
\(\int (u \pm v)\,dx = \int u\,dx \pm \int v\,dx\)

Exponential & Logarithmic Integrals

\(\int e^x dx = e^x + C\)
\(\int a^x dx = \frac{a^x}{\ln a} + C\)
\(\int e^{mx} dx = \frac{1}{m}e^{mx} + C\)
\(\int \ln x\,dx = x\ln x - x + C\)

Basic Trigonometric Integrals

\(\int \sin(ax)\,dx = -\frac{1}{a}\cos(ax) + C\)
\(\int \cos(ax)\,dx = \frac{1}{a}\sin(ax) + C\)
\(\int \sec^2 x\,dx = \tan x + C\)
\(\int \csc^2 x\,dx = -\cot x + C\)
\(\int \sec x \tan x\,dx = \sec x + C\)
\(\int \csc x \cot x\,dx = -\csc x + C\)

Substitution-Type Integrals

These are obtained using substitution.

\(\int \frac{f'(x)}{f(x)} dx = \ln|f(x)| + C\)
\(\int f(x)^n f'(x)\,dx = \frac{f(x)^{n+1}}{n+1} + C\)
\(\int \frac{f'(x)}{\sqrt{f(x)}} dx = 2\sqrt{f(x)} + C\)
\(\int (ax+b)^n dx = \frac{(ax+b)^{n+1}}{a(n+1)} + C\)

Integration by Parts

Used when the integrand is a product of two functions.

\[ \int u\,dv = uv - \int v\,du \]
LIATE Rule (Choosing u):
Logarithmic → Inverse Trigonometric → Algebraic → Trigonometric → Exponential

Inverse Trigonometric Forms

\(\int \frac{dx}{1+x^2} = \tan^{-1}x + C\)
\(\int \frac{dx}{a^2+x^2} = \frac{1}{a}\tan^{-1}\left(\frac{x}{a}\right) + C\)
\(\int \frac{dx}{a^2-x^2} = \frac{1}{2a}\ln\left|\frac{a+x}{a-x}\right| + C\)
\(\int \frac{dx}{x^2-a^2} = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C\)
\(\int \frac{dx}{x\sqrt{x^2-1}} = \sec^{-1}x + C\)

Square Root Forms

\(\int \sqrt{a^2-x^2}\,dx = \frac{x}{2}\sqrt{a^2-x^2} + \frac{a^2}{2}\sin^{-1}\left(\frac{x}{a}\right) + C\)
\(\int \sqrt{x^2+a^2}\,dx = \frac{x}{2}\sqrt{x^2+a^2} + \frac{a^2}{2}\ln\left|x+\sqrt{x^2+a^2}\right| + C\)
\(\int \sqrt{x^2-a^2}\,dx = \frac{x}{2}\sqrt{x^2-a^2} - \frac{a^2}{2}\ln\left|x+\sqrt{x^2-a^2}\right| + C\)

Reduction Formulas

\(\int \sin^n x\,dx = -\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x\,dx\)
\(\int \cos^n x\,dx = \frac{1}{n}\cos^{n-1}x\sin x + \frac{n-1}{n}\int \cos^{n-2}x\,dx\)
\(\int x^n e^x dx = x^n e^x - n\int x^{n-1} e^x dx\)
