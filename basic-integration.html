Integration (Intro)

Fundamental Rules

  • ∫ k dx = kx + C
  • ∫ xⁿ dx = xⁿ⁺¹ / (n+1) + C   (n ≠ −1)
  • ∫ 1/x dx = ln|x| + C
  • ∫ (u ± v) dx = ∫ u dx ± ∫ v dx

Exponential & Logarithmic Integrals

  • ∫ eˣ dx = eˣ + C
  • ∫ aˣ dx = aˣ / ln a + C
  • ∫ e^(mx) dx = (1/m)e^(mx) + C
  • ∫ ln x dx = x ln x − x + C

Basic Trigonometric Integrals

  • ∫ sin(ax) dx = −(1/a) cos(ax) + C
  • ∫ cos(ax) dx = (1/a) sin(ax) + C
  • ∫ sec²x dx = tan x + C
  • ∫ csc²x dx = −cot x + C
  • ∫ sec x tan x dx = sec x + C
  • ∫ csc x cot x dx = −csc x + C

Substitution-Type Integrals

These are obtained using substitution.

  • ∫ f'(x)/f(x) dx = ln|f(x)| + C
  • ∫ f(x)ⁿ f'(x) dx = f(x)ⁿ⁺¹/(n+1) + C
  • ∫ f'(x)/√f(x) dx = 2√f(x) + C
  • ∫ (ax+b)ⁿ dx = (ax+b)ⁿ⁺¹ / [a(n+1)] + C

Integration by Parts

Used when the integrand is a product of two functions.

∫ u dv = u·v − ∫ v du

LIATE Rule (Choosing u):
Logarithmic → Inverse Trigonometric → Algebraic → Trigonometric → Exponential
Choose u from the earliest possible category.

Inverse Trigonometric Forms

  • ∫ dx / (1 + x²) = tan⁻¹x + C
  • ∫ dx / (a² + x²) = (1/a) tan⁻¹(x/a) + C
  • ∫ dx / (a² − x²) = (1/2a) ln|(a+x)/(a−x)| + C
  • ∫ dx / (x² − a²) = (1/2a) ln|(x−a)/(x+a)| + C
  • ∫ dx / (x√(x²−1)) = sec⁻¹x + C

Square Root Forms

  • ∫ √(a² − x²) dx = (x/2)√(a²−x²) + (a²/2) sin⁻¹(x/a) + C
  • ∫ √(x² + a²) dx = (x/2)√(x²+a²) + (a²/2) ln|x+√(x²+a²)| + C
  • ∫ √(x² − a²) dx = (x/2)√(x²−a²) − (a²/2) ln|x+√(x²−a²)| + C

Reduction Formulas

  • ∫ sinⁿx dx = −(1/n) sinⁿ⁻¹x cos x + (n−1)/n ∫ sinⁿ⁻²x dx
  • ∫ cosⁿx dx = (1/n) cosⁿ⁻¹x sin x + (n−1)/n ∫ cosⁿ⁻²x dx
  • ∫ xⁿeˣ dx = xⁿeˣ − n ∫ xⁿ⁻¹eˣ dx
